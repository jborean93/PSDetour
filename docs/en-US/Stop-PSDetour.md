# Stop-PSDetour

## SYNOPSIS

Stops a PSDetour session and removes all active hooks.

## SYNTAX

```
Stop-PSDetour [<CommonParameters>]
```

## DESCRIPTION

Stops the active PSDetour session that was started by `Start-PSDetour`.

## EXAMPLES

### Example 1

```powershell
PS C:\> Stop-PSDetour
```

Stops the detour session.
